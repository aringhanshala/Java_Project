System Components

📝

Log Collection

Intercepts all HTTP requests and user actions. Captures IP address, timestamp, username, endpoint, response code, and user-agent for every event.

🔍

Log Parser

Java regex-based parser extracts and structures log data. Identifies log types: AUTH, ACTIVITY, SYSTEM and classifies them into the database.

⚠️

Threat Detection

Rule-based engine detects Brute Force, SQL Injection, XSS, Path Traversal, and DDoS patterns using threshold analysis and regex matching.

🗄️

Data Storage

All logs and alerts stored in H2 database via Spring Data JPA. Structured schema with indexed queries for fast analytics retrieval.

📊

Admin Dashboard

Real-time dashboard showing log distribution, threat alerts with severity levels, top suspicious IPs, and system health metrics.

🧪

Attack Simulator

Built-in attack simulation tool to trigger Brute Force, SQL Injection, XSS, and Path Traversal scenarios for testing the detection engine.
